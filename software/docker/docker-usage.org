#+Title: Docker 사용법
* 설치
=curl -fsSL https://get.docker.com/ | sudo sh=

* 설치확인
=sudo docker version=

* 컨테이너 실행
=docker run [이미지명]=


* 컨테이너 목록 확인하기
=docker ps [OPTIONS]=

* 컨테이너 중지하기 
=docker stop [OPTIONS] CONTAINER [CONTAINER...]=

참고로 도커 ID는 중복되지 않는다면 처음 몇 자만 입력해도 된다.

* 로그 보기
=docker logs [OPTIONS] CONTAINER=


* Dockerfile이 있을 때 이미지 만들기
Dockerfile이 있는 디렉토리로 이동 후 

=docker build -t [이미지명] .=
